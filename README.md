🚔 Crime Lens: Interactive Crime Analysis Dashboard
📌 Overview

Crime Lens is an interactive crime data analysis dashboard built using Power BI.
The project focuses on transforming raw crime datasets into meaningful insights through data cleaning, analysis, and visualization. It helps in identifying crime trends, hotspots, and patterns to support better decision-making.

🎯 Objectives
Analyze overall crime trends over time
Identify most frequent crime types
Detect high-crime locations (hotspots)
Study victim demographics and case status
Understand time-based crime patterns

📂 Dataset
Source: Government open data (data.gov.in)
Contains:
Crime type
Date & time
Location (latitude & longitude)
Victim details
Weapon used
Case status

⚙️ Data Preprocessing
Removed duplicate records
Handled missing/null values
Standardized column names
Formatted date & time for analysis
Cleaned categorical data (crime type, gender, etc.)
Validated geographical coordinates

📊 Dashboard Features
📈 KPI Cards: Total crimes, solved cases, pending cases
🗺️ Map Visualization: Crime hotspots using latitude & longitude
📊 Charts:
Crime type distribution
Time-based trends (year/month)
Gender-wise victim analysis
Weapon usage insights
🎛️ Interactive Filters (Slicers) for dynamic analysis

🚀 Key Insights
Identified peak crime periods based on time analysis
Highlighted top crime categories contributing to overall crime rate
Detected high-risk locations using map-based visualization
Found demographic patterns in victim data
Revealed a significant percentage of unresolved cases
